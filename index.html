Loqu

Loqu is a translator made with artificial intelligence.

Discover More

Have you faced trouble in communicating with people in your locality?

Loqu is a project that aims to help people who speak a different language from those in their locality. Loqu is made using trained artificial intelligence models, which helps in translating as it can remove the need of people being translators, and people can directly go to Loqu and translate what needs to be told. Furthermore, trained artificial intelligence models have the data needed to translate one part of a speech to a different language, which helps in communicating with the other person. Loqu takes in speech from what is told and gives out an audio file that can be played to the person you are trying to communicate with.

Product

Loqu is on Google Colaboratory, a platform on which artificial intelligence algorithms can be shared.

Instructions: To use Loqu, record a voice memo with the words you intend to translate. Then, upload it to the Google Colab runtime. Run all the lines of code except for the last one, which you can change the languages in. Put the file path in the first part, the initial language in the second, and the language you intend to translate the words into in the third. It will generate an audio file in the files section, which you can download and play the translated version to whoever you wish to communicate your message to.
